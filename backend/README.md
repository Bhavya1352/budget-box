# BudgetBox Backend

## Setup

```bash
cd backend
npm install
npm run dev
```

## API Endpoints

- `POST /budget/sync` - Sync budget data
- `GET /budget/latest/:userId` - Get latest budget
- `POST /auth/login` - Demo login
- `GET /health` - Health check
